== Description ==

Abre de forma suave com javascript um conteúdo de um post ou pagina.

== Installation ==

1. Upload `peq-popup-adobe-spry.php` to the `/wp-content/plugins/` directory
2. Use short code [peqPopup id="139" px="50" py="50" efeito="Blind" duracao="1500"  mouse="false"]vai que da rapa[/peqPopup]

Exemplo

caso utilize o editor ckeditor adicionar a linha de comando ao arquivo:
ckeditor-for-wordpress/ckeditor.config.js
config.entities = false;
